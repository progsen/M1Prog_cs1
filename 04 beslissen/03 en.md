## || OF

- lees:
```

- || werkt zoals de && maar volgens iets andere regels:
    - ALS links of rechts true is DAN is het resultaat true
        > false || true => true
        > true || false => true
    - als beide waar zijn, DAN true
        > true || true => true
    - als beide false zijn dan false 
        > false || false => false
```

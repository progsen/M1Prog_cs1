## clone 

- type het volgende in je terminal (druk NIET op enter)
    ```
    git clone
    ```
- plak de url erachter, dan krijg je zoiets:
    ```
    git clone https://github.com/progsen/M1Prog_cs1.git
    ```
- druk op enter
